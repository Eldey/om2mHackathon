# om2mHackathon
This is the respository of the om2mHackathon @Wollongong.


## Quick start

### Requirement

```powershell
JAVA Runtime Environment (JRE) > 1.8
```

### Install on Windows

```powershell
Unzip the file x86_64.zip 
Launch file start.bat
```

### Install on Unix

```powershell
Unzip the file x86_64.zip 
Launch file start.sh
```
